Hey, and welcome to Tender!

Tender is a flexible and beginner-friendly programming language that’s both powerful and simple. Whether you’re building something serious or just having fun, Tender is designed to help you get up and running fast.

Let’s dive in and explore the basics together!

Variables

In Tender, you can declare variables easily using :=, or with var if you want more control. No need to specify types—Tender figures them out for you. 

name := "Alice"       // String  
age := 30             // Integer  
pi := 3.14            // Float  
is_admin := false     // Boolean  
var year = bigint(2024)       // Big integer  
var c = complex(pi, age)      // Complex number

Reassigning Variables

Variables are dynamically typed, so you can change their type later if you want.

age = "Thirty"  // Now 'age' is a string  
println(age)    // Output: Thirty

Arrays and Maps

Arrays

Arrays are ordered lists. You can put anything in them.

fruits := ["Apple", "Banana", "Cherry"]  
println(fruits[0])  // Output: Apple

Maps

Maps are like dictionaries—key-value pairs.

person := {name: "Alice", age: 30, is_admin: true}  
println(person["name"])  // Output: Alice

Control Flow

If/Else

Tender uses a clean if/else syntax. You can even initialize variables inside the if.

if age := 30; age > 18 {
    println("Adult")
} else {
    println("Minor")
}

Loops

For Loop

numbers := [1, 2, 3, 4]
for i := 0; i < len(numbers); i++ {
    println(numbers[i])
}

For-Each

fruits := ["mango", "apple", "banana"]
for i, fruit in fruits {
    if fruit == "apple" {
        continue
    } else if i == len(fruits) {
        break
    }
    println(fruit)
}

For Condition

for true {
    println("looping forever?")
}

For Infinity

for {}

Functions

Define a function using fn. It can take arguments and return stuff.

greet := fn(name) {
    return "Hello, " + name
}
println(greet("Alice"))

You can also declare functions like this:

fn greet(name) {
    return "Hello, " + name
}

Passing Functions

each := fn(arr, action) {
    for x in arr {
        action(x)
    }
}
each([1, 2, 3], fn(n) {
    println(n)
})

Closures

Closures let you create private variables inside functions:

counter := fn() {
    count := 0
    return fn() {
        count += 1
        return count
    }
}
next := counter()
println(next())  // 1  
println(next())  // 2

Recursion & Tail-Call Optimization

factorial := fn(n, acc) {
    if n == 0 {
        return acc
    }
    return factorial(n - 1, acc * n)
}
println(factorial(5, 1))  // Output: 120

Slicing

str := "hello world"
println(str[0:5])  // hello

arr := [10, 20, 30, 40, 50]
println(arr[1:4])  // [20, 30, 40]

Type Conversion

str_num := string(42)   // "42"  
int_num := int("99")    // 99  
float_num := float(3)   // 3.0  
bool_val := bool(1)     // true
